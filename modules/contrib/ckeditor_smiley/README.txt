INTRODUCTION
------------
This module enables the Smiley plugin from CKEditor.com in your WYSIWYG. This
plugin provides a set of emoticons for inserting into the editor via a dialog
window.

INSTALLATION
------------
1. Download the plugin from http://ckeditor.com/addon/smiley.
2. Place the plugin in the root libraries folder (/libraries).
3. Enable CKEditor Smiley in the Drupal admin.
4. Configure your WYSIWYG toolbar to include the button.

REQUIREMENTS
------------
CKEditor Module (Core)
